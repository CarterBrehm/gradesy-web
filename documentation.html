Overview

Login Page

The bulk of the login page is done in stock HTML with jQuery Mobile applied on top with the nativedroid UI skin. There's a small script at the top of the page that takes the details entered by the user, URIencodes them, and passes them on to the next page.

Grades Page (Main Page)

On page load, a PHP script runs (using the passed credentials) and emulates the users browser to retrieve the main schedule page on the Esu3 website. From there, it generates HTML (in the form of multiple jQuery Mobile pages) and then passes it to jQuery Mobile, which then forms the main structure of the page. This page is also themed by nativedroid.
